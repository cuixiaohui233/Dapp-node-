其次，用户在两天之后将不能修改自己的数据，管理员无论何时都可以修改。

我们可以将这个分为两个：

    读权限：用户-> 只可以读自己的

           管理员-> 全部
           
           合约方法需要知道读这个合约的人是谁

    写权限：用户-> 两天之内可以修改自己的数据
    
           管理员-> 永远可以
           
           合约方法需要知道写合约的人是谁，以及这个人的写入时间
